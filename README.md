## Description
#####
#### 전통시장의 밀키트를 DIY 형태로 집에서 배달시켜 먹을 수 있도록 하기 위한 플랫폼입니다.
#### Front는 React.js , Backend는 django를 이용해 개발하였습니다.
#### React.js에서 axios를 이용해서 서버의 내용들을 불러왔습니다.

#####
#####
